# **App Name**: ING Belgium Online

## Core Features:

- Login Page: Secure login page to authenticate users.
- Dashboard: Dashboard displaying account balances, transaction history, and quick actions.
- Card Page: Display user's card details (mock data) in a visually appealing manner.
- Transfer Page: Allow users to initiate wire transfers to other accounts.
- RIB Page: Display user's bank account details (RIB).
- Logo Integration: Fetch the ING Belgium Logo from the provided URL: https://imgur.com/WWZ10oQ.

## Style Guidelines:

- Primary color: Deep orange (#F2711D), mirroring ING's branding.
- Background color: Light beige (#F5F5DC), for a soft, inviting feel.
- Accent color: Complementary vibrant yellow (#FFEA00), used sparingly for highlights and action items.
- Body and headline font: 'Inter', sans-serif, offering a clean, modern, and trustworthy aesthetic.
- Use simple, clean icons for navigation and actions, sourced from a consistent icon set (e.g., FontAwesome or Material Icons).
- Clean, card-based layout with ample white space to avoid clutter.
- Subtle transitions and animations for page navigation and data loading to improve user experience.
